## 📘 LeetCode 9 - Palindrome Number

### 💡 Idea

Reverse the digits of the number and compare the reversed value with the original number to check if it is a palindrome.

---

### 🧠 Approach

* Store the original number
* Return false if the number is negative
* Extract digits one by one using modulo operation
* Build the reversed number
* Compare reversed number with original number

---

### ⏱ Complexity

* **Time:** O(log n)
* **Space:** O(1)

---

### 🧩 Pattern

Math Manipulation (Number Reversal)
