## 📘 LeetCode 344 - Reverse String

### 💡 Idea

Swap characters from both ends of the array until the middle is reached, reversing the string in-place.

---

### 🧠 Approach

* Use two pointers: one at start and one at end
* Swap characters at both pointers
* Move pointers inward after each swap
* Continue until pointers meet or cross

---

### ⏱ Complexity

* **Time:** O(n)
* **Space:** O(1)

---

### 🧩 Pattern

Two Pointers (In-place Reversal)
